# 📊 Detecter des faux billets avec python – ONCFM

Bienvenue sur la page consacrée au projet **Détecter des faux billes avec Python**. 

Ce projet illustre mes compétences en **Analyse de données, sélection et utilisation d'algorithme en respectant une problématique métier ** à travers des livrables concrets.


Pour mener à bien ce projet j'ai : 

* Exploré et visualisé les données avec Python
* Préparé et nettoyé le jeu de données
* Construit et comparé plusieurs modèles de classification
* Évaluer la performance de chaque modèle et choisit le plus efficace
* Identifié les critères qui permettent de repérer un faux billet
* Présenté clairement les résultats et démontré l'efficacité du modèle sur un jeu de données inconnues.

---

## 🧭 Contexte

**ONCFM**, l'organisation nationale de lutte contre le faux monnayage souhaite mettre en place des méthodes d'identifications de faux billets plus rapides, en utilisant une application de machine learning en testant différents algorithmes recommandés par l'agence européenne (EMV) en charge du faux monnayage.

---

## 🎯 Objectif

Determiner lequel algorythme préconnisé est le plus performant dans la détection des faux billets tout en limitant les erreurs concernant les vrais billets :  

- K-means  
- Regression logistique  
- KNN  
- Random Forest
